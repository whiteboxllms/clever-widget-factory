# Clever Widget Factory - Product Overview

## Purpose
Clever Widget Factory is a comprehensive asset management and accountability system designed to track tools, parts, actions, missions, and organizational activities. It provides real-time visibility into asset usage, team accountability, and operational workflows.

## Value Proposition
- **Asset Accountability**: Track tool checkouts, part inventory, and asset history with complete audit trails
- **Action Management**: Document and score team actions with strategic attributes (proactive vs reactive, documentation quality)
- **Mission Coordination**: Organize work into missions with linked actions and issues
- **Exploration System**: Capture experimental work and promote successful patterns into organizational policies
- **AI-Powered Search**: Semantic search across tools and parts using AWS Bedrock embeddings
- **Analytics Dashboard**: Visualize team performance, inventory trends, and strategic metrics

## Key Features

### Asset Management
- **Tool Tracking**: Check in/out tools with intended usage, expected return dates, and notes
- **Parts Inventory**: Track parts with quantities, locations, sellable status, and reorder points
- **Asset History**: Complete audit trail of all asset changes and transactions
- **Combined Asset View**: Unified interface for browsing both tools and parts

### Action & Mission System
- **Action Tracking**: Document work with rich text descriptions, photos, and strategic scoring
- **Mission Management**: Group related actions into missions with templates and progress tracking
- **Issue Reporting**: Link issues to actions and missions with resolution workflows
- **Action Scoring**: Rate actions on strategic attributes (proactive/reactive, documentation quality, etc.)

### Exploration & Policy System
- **Exploration Data Collection**: Capture experimental code, metrics, and observations
- **Exploration Status Workflow**: Draft → In Progress → Under Review → Completed/Abandoned
- **Policy Promotion**: Convert successful explorations into organizational policies with AI assistance
- **Policy Linking**: Associate actions with existing policies for compliance tracking

### Search & Discovery
- **Semantic Search**: Natural language search across tools and parts using AWS Bedrock Titan embeddings
- **Advanced Filtering**: Filter assets by status, location, category, and custom attributes
- **Sari-Sari Chat**: AI-powered conversational interface for querying organizational data

### Analytics & Reporting
- **Action Analytics**: Proactive vs reactive ratios, documentation quality trends
- **Inventory Analytics**: Stock levels, usage patterns, reorder alerts
- **User Activity**: Individual and team performance metrics
- **Timeline Views**: Historical activity tracking across all entities

## Target Users

### Field Workers
- Check out/in tools for daily work
- Document actions with photos and notes
- Report issues and track resolutions
- View assigned missions and tasks

### Team Leaders
- Monitor team activity and performance
- Review and score team actions
- Manage missions and assign work
- Track tool availability and usage

### Organization Administrators
- Manage organization settings and members
- Configure policies and workflows
- Access analytics and reports
- Oversee multi-organization operations

### Super Administrators
- Manage multiple organizations
- Configure system-wide settings
- Access all data across organizations
- Perform system maintenance

## Use Cases

### Daily Operations
- Worker checks out tools at start of day, documents work actions, checks in tools at end of day
- Team leader reviews completed actions, provides scores and feedback
- Inventory manager monitors part levels and creates reorder actions

### Project Management
- Create mission for new project with action templates
- Assign team members and track progress
- Link related issues and explorations
- Generate completion reports with analytics

### Process Improvement
- Capture experimental approaches in explorations
- Collect metrics and observations during trials
- Review exploration results with team
- Promote successful patterns to organizational policies

### Inventory Management
- Track part quantities and locations
- Set reorder points and receive alerts
- Document part usage in actions
- Analyze consumption patterns

### Compliance & Audit
- Link actions to organizational policies
- Track policy agreement timestamps
- Generate audit trails for all asset changes
- Review historical activity timelines
